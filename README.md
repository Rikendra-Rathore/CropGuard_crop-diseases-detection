📌 Overview

CropGuard is an AI-powered crop disease detection system designed to help farmers identify plant diseases quickly and accurately. The system analyzes crop leaf images using machine learning techniques and predicts possible diseases.

In addition to disease detection, CropGuard includes an AI chatbot that answers user questions related to crop diseases, symptoms, and treatments. The goal of the system is to help farmers take timely action to protect crops and improve agricultural productivity.

❗ Problem Statement

Farmers often face significant crop losses due to plant diseases. Early detection is difficult because many farmers do not have access to agricultural experts or proper diagnostic tools. Traditional disease identification methods rely on manual observation, which can be slow and sometimes inaccurate.

CropGuard addresses this problem by providing an automated system that can quickly detect crop diseases from plant images and assist farmers with useful recommendations.

💡 Solution Offered

CropGuard provides a simple web-based platform where users can upload crop leaf images. The system processes the image using machine learning models to detect and classify plant diseases. After prediction, the system displays the disease name and provides preventive measures or treatment suggestions.

The integrated chatbot further assists users by answering queries related to crop diseases and farming practices.

📊 System Workflow

User uploads a crop leaf image.

Image preprocessing is performed.

The machine learning model analyzes the image.

The system predicts the crop disease.

Results and preventive recommendations are displayed.

The chatbot answers additional user questions.
